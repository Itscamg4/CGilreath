InstaCross

Client-Side Instagram Relationship Analyzer

Complete
Vanilla JavaScriptRegexFileReader APIClient-Side Only

Overview

A privacy-first, zero-server web utility designed to help users clean up their social graph. By leveraging Instagram's "Download Your Data" files, the tool instantly extracts, normalizes, and compares follower data to reveal non-reciprocal accounts and mutual connections — entirely inside the browser, with nothing ever sent to a server.

Technical Highlights

Zero-Server Footprint Built entirely in vanilla HTML, CSS, and JS. Because social data is highly sensitive, the application processes files strictly within the browser environment via the FileReader API — guaranteeing total data privacy.
Polymorphic Data Parsing Written with resilient Regex patterns that gracefully handle changing data shapes. It seamlessly extracts usernames whether the user uploads raw HTML source code or pastes a plain-text list of names.
Algorithmic Efficiency Built to handle thousands of rows of data instantly by converting array structures into hash-ready JavaScript Set structures, preventing UI lag during heavy string manipulation.
