Arduino Security System

Active June 2026
ESP8266Ultrasonic SensorFlaskTkinterSMTP

Overview

A WiFi-connected security alarm built around an ESP8266 microcontroller and an ultrasonic sensor. The system pairs low-level hardware alerting with a desktop control application, so it works standalone but can also notify me remotely the moment it detects motion.

Highlights

Resilient Networking Implemented automated WiFi reconnection logic so the alarm re-establishes its connection and resumes monitoring after any network drop, without manual intervention.
Local + Remote Alerting Combined local hardware alerts (triggered directly off the ultrasonic sensor) with HTTP-based data dispatching, so the system reports state changes to a backend in real time.
Multi-Threaded Desktop Interface Built a Flask backend and a Tkinter desktop interface running on separate threads, keeping the UI responsive while continuously polling sensor and network state.
Smart Notification Cooldown Integrated an SMTP email notification system with a custom cooldown algorithm, preventing alert flooding while still surfacing every meaningful event.
